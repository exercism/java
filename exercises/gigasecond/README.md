# Gigasecond

Calculate the moment when someone has lived for 10^9 seconds.

A gigasecond is 10^9 (1,000,000,000) seconds.

# Running the tests

You can run all the tests for an exercise by entering

```sh
$ gradle test
```

in your terminal.
