# Word Count

Given a phrase, count the occurrences of each word in that phrase.

For example for the input `"olly olly in come free"`

```text
olly: 2
in: 1
come: 1
free: 1
```

# Running the tests

You can run all the tests for an exercise by entering

```sh
$ gradle test
```

in your terminal.
