# Complex Numbers

A complex number is a number in the form `a + b * i` where `a` and `b` are real and `i` satisfies `i^2 = -1`.

Assume the programming language you are using does not have an implementation of complex numbers.


To run the tests:

```sh
$ gradle test
```
